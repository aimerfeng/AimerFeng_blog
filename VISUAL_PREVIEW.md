## Logo 设计详解

```
     ╱─────╲
    ╱       ╲
   ╱    ●    ╲     ← 顶部节点 (AI数据点)
  ╱           ╲
 ●             ●   ← 左右节点
 │             │
 │      A      │   ← 字母 A (AimerFeng)
 │     ╱ ╲     │
 │    ╱   ╲    │
 │   ╱─────╲   │
 ●             ●   ← 左右节点
  ╲           ╱
   ╲    ●    ╱     ← 底部节点
    ╲       ╱
     ╲─────╱

六边形 = 区块链
字母 A = AimerFeng
6个点 = AI/数据节点
```

## 交互状态

### 卡片悬停效果
```
正常状态:
┌─────────────┐
│   Content   │  opacity: 0.1
└─────────────┘  shadow: 8px

悬停状态:
┌─────────────┐
│   Content   │  opacity: 0.15
└─────────────┘  shadow: 12px
      ↑ 上浮 2px
```

### 主题切换动画
```
亮色 → 暗色

🌞 ──────────────→ 🌙
    (0.3s 过渡)

所有颜色平滑过渡:
- 背景色
- 文字色
- 玻璃效果
- 渐变颜色
```

## 特殊效果

### 边框高光
```
┌─────────────────────────┐
│ ✨ ─────────────── ✨   │ ← 顶部渐变高光
│                         │   (透明→白色→透明)
│      Content            │
│                         │

```

### 滚动按钮
```
滚动 < 300px:  隐藏 (opacity: 0)
滚动 > 300px:  显示 (opacity: 0.3)

┌───┐
│ ↑ │  ← 圆形按钮
└───┘    悬停时 opacity: 1
```
