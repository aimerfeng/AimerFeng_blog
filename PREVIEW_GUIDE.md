# 🎨 预览指南 - AimerFeng Blog

## 快速启动

### 方法 1: 使用 pnpm (推荐)

```bash
# 1. 安装 pnpm (如果还没安装)
npm install -g pnpm

# 2. 安装依赖
cd aimerfeng-blog
pnpm install

# 3. 启动开发服务器
pnpm run dev
```

然后在浏览器中打开 `http://localhost:3333`

### 方法 2: 使用 npm

由于项目使用了 pnpm catalog 特性，建议使用 pnpm。

## 🎨 视觉效果预览

### 亮色主题
```

渐变: 柔和的蓝、紫、粉、绿色
```

### 暗色主题

## 🎭 交互效果

1. **卡片悬停**: 
   - 背景透明度增加
   - 轻微上浮动画 (translateY: -2px)
   - 阴影加深

2. **主题切换**: 
   - 点击主题切换按钮
   - 平滑过渡动画
   - 所有颜色自动适配

3. **响应式设计**:
   - 桌面端: 完整导航和社交链接
   - 移动端: 简化图标导航

4. **无障碍支持**:
   - 支持 prefers-reduced-motion
   - 键盘导航友好
   - 语义化 HTML

## 📱 响应式断点

- **桌面**: > 768px - 完整布局
- **平板**: 768px - 768px - 适配布局
- **手机**: < 768px - 移动优化

## 🎬 动画细节

### 背景渐变动画
- **动画时长**: 60秒 / 80秒
- **动画类型**: 旋转 + 缩放
- **缓动函数**: ease / ease-in-out
- **方向**: 正向 / 反向

### 玻璃卡片
- **模糊度**: 10px backdrop-filter
- **过渡**: 0.3s ease
- **阴影**: 多层次阴影效果

## 🔧 技术栈

- **框架**: Vue 3 + Composition API
- **构建**: Vite + vite-ssg (静态站点生成)
- **样式**: UnoCSS + 自定义 CSS
- **内容**: Markdown + Shiki 语法高亮
- **类型**: TypeScript
